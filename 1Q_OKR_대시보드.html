프로토타입 구축

이벤트 디자인의 반복 요소를 구조화/모듈화하여 자동화 기반을 구축하고, 이벤트 제작 생산성을 높이는 것이 목표입니다.

✓ 1분기 OKR 전체 달성 완료 (KR1 100%)
🎯 Objective & Key Results
OBJECTIVE

이벤트 디자인의 반복 요소를 구조화/모듈화하여 자동화 기반을 구축하여 생산성을 높인다.

  • 1
    이벤트 공통 영역 프로토타입 구현 반복 사용되는 이벤트 공통 영역(적립금, 쿠폰, 도서 리스트 등) 구조 모듈 정의. 공통 영역 프로토타입 코드 구현 및 실제 이벤트 적용을 통한 동작 검증. 공통 영역 적용을 통해 이벤트 제작 소요 시간 약 30% 단축 및 MD 요청 기반 수정 업무 소요 시간 50% 이상 단축.
    ✓ KR1 완료
🔍 배경 및 문제
  • • 이벤트 페이지 제작 시 적립금, 쿠폰, 도서 리스트, 증정품, 댓글 영역이 이벤트마다 반복적으로 재작성
  • • 이전 파일 또는 매뉴얼을 참고해 복붙하는 방식 사용
  • • MD가 직접 수정할 경우 태그 누락/닫힘 오류로 페이지가 깨지는 문제 반복 발생
  • • 디자이너가 작성하더라도 해당 코드를 찾고 수정하는 시간이 비효율적으로 소요
