2. 상단이미지 (3단계 위자드)

1타이틀 방식 선택

3가지 중 선택합니다:

  • 텍스트로 직접 입력 — 배경 이미지 위에 h태그로 타이틀을 코드에 입력
  • 이미지에 타이틀 포함 — AI가 타이틀 글자까지 이미지로 생성
  • 타이틀 없이 배경만 — 순수 배경 이미지만 사용
2AI 이미지 프롬프트 생성

이벤트 주제, 분위기, 컬러, 스타일 등을 선택하면 ChatGPT/Gemini에 바로 붙여넣을 수 있는 프롬프트가 자동 생성됩니다.

  • 🖥️ PC용 복사 / 📱 모바일용 복사 버튼으로 각각 복사
  • 타이틀 방식에 따라 프롬프트 내용이 자동 조정됨
3이미지 설정 & 코드 추가

이미지 제작이 완료되면:

  • 모바일/PC 이미지 URL 입력 (분기별 경로 자동 생성됨)
  • 이미지에서 추출 버튼으로 배경색 자동 추출 가능
  • 색상 미리보기 박스를 클릭하면 컬러피커로 직접 선택 가능
  • 텍스트 타이틀 선택 시: 4가지 스타일(직접 노출/원형 글래스/블러 라운드/이미지 아래) 중 선택
  • PC 그라데이션 페이드 체크 시 양옆이 배경색으로 자연스럽게 이어짐
이미지 크기를 자동 감지하여 background-size가 자동 설정됩니다. 배경색과 타이틀 색상은 컬러피커에서 변경하면 미리보기에 실시간 반영됩니다.

3. 텍스트 표기법

타이틀이나 신간 제목 입력 시 특수 표기로 줄바꿈과 강조를 적용할 수 있습니다.

{텍스트}<span class="t_pink">핑크색 강조

→ <span class="br">외국어가</span> <span class="br">나에게로 옵니다! <span class="t_pink">특별판</span></span>

5. 미리보기NEW

우측 패널 상단의 코드 / 미리보기 탭으로 전환할 수 있습니다.

  • 알라딘 실서버 CSS/JS를 로드하여 실제와 유사한 미리보기 제공
  • Swiper 슬라이더 실제 동작
  • 서버 컨트롤(uc1 태그)은 더미 플레이스홀더로 표시
  • 배경색/타이틀 색상 변경 시 미리보기에 실시간 반영

7. 패널 조절

좌측 입력 패널과 우측 코드/미리보기 패널 사이의 경계선을 드래그하면 좌우 비율을 자유롭게 조절할 수 있습니다. (최소 500px ~ 최대 1200px)

8. 이미지 경로 자동 생성

현재 날짜 기준으로 분기별 이미지 경로가 자동 생성됩니다.

분기경로
1분기 (1~3월)//image.aladin.co.kr/img/events/book/{년도}/01/
2분기 (4~6월)//image.aladin.co.kr/img/events/book/{년도}/02/
3분기 (7~9월)//image.aladin.co.kr/img/events/book/{년도}/03/
4분기 (10~12월)//image.aladin.co.kr/img/events/book/{년도}/04/
경로가 자동 입력되니 파일명만 수정하면 됩니다.

9. 주의사항

서버 컨트롤(<uc1:...>)은 알라딘 서버에서만 동작합니다. 미리보기에서는 더미로 표시됩니다.
쿠폰 번호(nType), 이벤트 ID, AuthorId 등은 실제 값으로 반드시 교체해야 합니다.
모든 모듈 추가가 끝나면 반드시 완성 코드 내보내기를 눌러 닫기 태그를 추가하세요.
