이벤트 공통 모듈 빌더 v1.1.2

증정품 · 도서리스트 · 댓글 · 쿠폰/적립금 코드를 자동으로 생성하는 도구

1. 시작하기

브라우저에서 HTML 파일을 열면 바로 사용 가능합니다. 별도 설치가 필요 없습니다.

1탭 선택

상단 4개 탭(증정품, 도서 리스트, 댓글, 쿠폰/적립금) 중 필요한 모듈을 선택합니다.

2타입 선택 & 입력

드롭다운에서 세부 타입을 선택하면 입력 폼이 나타납니다. 예시 값이 미리 입력되어 있으니 필요한 부분만 수정합니다.

3코드 추가

하단 + 현재 탭 내용 추가 버튼을 클릭하면 우측 코드 에디터에 코드가 추가됩니다.

4복사 & 붙여넣기

📋 전체 복사 버튼으로 코드를 복사한 뒤, .ascx 파일에 붙여넣습니다.

2. 지원 모듈

🎁 증정품 (2개 타입)

타입설명
단일 증정품 (팝업+정보형)슬라이더 이미지 + 자세히보기 팝업 + 증정 조건 + 유의사항
증정품 (하단 썸네일형)슬라이더 + 하단 썸네일 리스트 (여러 증정품 표시)
슬라이더 이미지: 이미지 주소를 입력하고 엔터를 치면 각각 하나의 슬라이드(div)로 자동 생성됩니다. 엔터로 구분하여 여러 장을 입력할 수 있습니다.
유의사항: 마찬가지로 엔터를 치면 각 줄이 하나의 li 항목으로 자동 변환됩니다.

📚 도서 리스트 (4개 타입)

타입설명
단일 도서 리스트CID 기반 자동 도서 목록
다중 도서 리스트 (탭)여러 CID를 탭으로 분리하여 표시
직접 도서 선택ItemID를 직접 입력하여 개별 도서 표시
랜덤 탭wa_RandomTab 서버 컨트롤 사용
직접 도서 선택: ItemID를 쉼표(,)로 구분하여 입력하면 각각 개별 도서 리스트로 생성됩니다. 예: 290210397, 291710358, 289175924

💬 댓글 (4개 타입)

타입설명
일반 댓글기본 댓글 폼 (EventComment v3)
상품 선택형댓글 작성 시 상품 선택 가능
저자 추천형저자 추천 기능 포함
이미지 첨부형이미지 업로드 기능 포함

🎟️ 쿠폰/적립금 (3개 타입)

타입설명
단일 쿠폰 (nType)하나의 쿠폰 번호로 다운로드
다중 쿠폰 (nTypeList)여러 쿠폰을 콤마로 구분하여 표시
기본 적립금적립금 표시 + 댓글 작성 링크 옵션
메인 타이틀: 엔터를 치면 줄바꿈이 적용됩니다. 각 줄이 span.br로 감싸져 모바일에서 줄바꿈 처리됩니다.
유의사항: 엔터를 치면 각 줄이 하나의 li 항목으로 자동 변환됩니다.

3. 기본 헤더

기본 헤더 포함 체크박스로 ASP.NET 헤더 코드(Register 지시문, CSS/JS 링크)를 포함하거나 제외할 수 있습니다.

처음 만드는 이벤트 파일이면 헤더를 포함하고, 기존 파일에 모듈만 추가하는 경우 체크 해제하세요.

4. 주의사항

생성된 코드의 서버 컨트롤(<uc1:...>)은 알라딘 서버에서만 동작합니다. 로컬에서는 미리보기가 불가합니다.
쿠폰 번호(nType), 이벤트 ID 등은 실제 값으로 반드시 교체해야 합니다.
