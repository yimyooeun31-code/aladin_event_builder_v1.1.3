코드 완료 & 복사
💡 미리보기 / 코드 탭 — 우측 패널 상단의 미리보기 · 코드 탭을 전환해 실시간 화면 확인과 생성된 코드를 오가며 확인 가능.

💡 기본 헤더 포함 — 기본 헤더 포함 체크 시 공통 헤더 코드(CSS·JS 링크 포함)가 자동 삽입됨. 최종 .ascx 파일 작성 시 필요.

💡 복사하기 — 우측 상단 복사하기 버튼을 누르면 현재까지 추가된 코드를 즉시 복사할 수 있음.

💡 코드완료 — 모든 모듈 추가 후 반드시 코드완료 버튼을 눌러야 </article> 닫기 태그가 포함된 최종 코드가 완성됨.

💡 줄바꿈 표기 — 타이틀 입력 시 /br2를 삽입하면 PC에서는 줄바꿈, 모바일에서는 한 줄로 처리됨. 반대로 /br은 모바일 줄바꿈, PC 한 줄.
미리보기에서 수정하기
  • 코드 추가 후 미리보기 패널에서 각 모듈 영역에 마우스를 올리면 수정 컨트롤이 나타납니다.
  • 상단이미지 · 신간알림 상단 — 내용 수정만 가능. 위아래 순서 이동은 지원하지 않습니다. 수정 버튼 클릭 → 입력 내용 변경 후 다시 코드 생성.
  • 일반 모듈 (도서 슬라이딩, 쿠폰, 댓글 등) — ↑ ↓ 버튼으로 순서 조정, 수정 버튼으로 내용 편집, 🗑 버튼으로 즉시 삭제.

이벤트 상단 디자인

화면 보기 ↗

이벤트 페이지 최상단 배너 영역.

종류
모바일 only 전용
모바일 only 템플릿 상단 이미지 수정. 타이틀 위치 조정 가능.
제작 순서
  1. PC 배경 이미지 준비 — 가로 1400px × 세로 440px. 좌우가 자연스럽게 보이도록 그라데이션 페이드 적용 가능.
  2. 이미지 URL 입력 후 이미지에서 추출 클릭 → 배경 이미지와 어울리는 배경색 자동 추출.
  3. 그라데이션 적용 체크 시 배경 좌우로 색상이 자연스럽게 페이드 처리됨.
  4. 다음 클릭 → 이벤트 타이틀 입력.
  5. 다음 클릭 → 타이틀 위치 스타일 선택.
    • 텍스트 직접 노출
    • 원형 글래스 박스
    • 블러 라운드 박스
    • 이미지 아래 별도 (기본)
  6. 위 3개 오버레이 스타일 선택 시 타이틀 색상 직접 변경 가능. 이미지 아래 별도 선택 시 색상 설정 비활성화.
  7. 스타일 선택 후 코드완료 클릭 → 코드 생성 완료.
신간알림 이벤트 스타일
신간 소개 이벤트 템플릿에 최적화된 상단 디자인 스타일.
제작 순서
  1. 소개할 책이 단권이면 단일 표지, 2권 이상이면 여러 권 선택 — 최대 4권까지 가능.
  2. 생성된 배경 이미지 주소 입력.
  3. 다크 오버레이 적용 체크 시 타이틀 가독성 향상 — 배경 이미지 밝기에 따라 선택.
  4. 다음 클릭 → 타이틀 텍스트 및 색상 변경 후 완료.
필요한 정보
  • 모바일 배경 이미지 URL
  • PC 배경 이미지 URL (신간알림 스타일)
  • 상단 배경색 — 이미지 위아래 여백 컬러
  • 제목·부제목 텍스트 (선택)

도서 슬라이딩

화면 보기 ↗

Swiper 기반 도서 가로 슬라이드 섹션. V1~V5 총 5가지 스타일.

버전특징
V1무한 자동 스크롤. 표지 이미지 URL 직접 입력
V2중앙 포커스 + 하단 도서 정보 카드. ItemID 입력
V3세로형 표지 + 진행 바. 표지 이미지 URL 직접 입력
V4전체화면 슬라이드 + 설명 텍스트. 표지 이미지 URL 직접 입력
V5무한 자동 스크롤 + ItemID 입력형
필요한 정보
  • 섹션 제목
  • 도서 표지 이미지 URL 또는 도서 ItemID — 버전에 따라 다름
  • 도서 제목·저자·설명 텍스트 — V2·V5만 해당

댓글 이벤트

화면 보기 ↗

알라딘 댓글 서버 컨트롤 삽입 섹션. 이벤트 목적에 따라 4가지 타입 중 원하는 옵션을 골라 사용할 수 있습니다.

댓글 타입
일반 댓글
기본 댓글 작성 폼. 자유롭게 의견을 남기는 일반적인 이벤트에 사용
상품 선택형
댓글 작성 시 응모할 상품을 직접 선택하는 옵션 포함. 경품 선택 이벤트에 적합
저자 추천형
추천 저자를 선택하거나 입력하는 옵션 포함. 저자 관련 이벤트·투표에 적합
이미지 첨부형
댓글에 이미지를 첨부할 수 있는 옵션 포함. 사진 인증 이벤트에 적합
제작 순서
  1. 원하는 댓글 타입을 선택합니다 — 일반 / 상품 선택 / 저자 추천 / 이미지 첨부 중 하나
  2. 이벤트 아이디(eventID)를 입력합니다 — 알라딘 이벤트 등록 후 발급되는 고유 아이디
  3. 섹션 제목을 입력합니다
  4. 댓글 작성 안내 문구를 입력합니다 선택
  5. 코드 추가 버튼을 누르면 댓글 모듈 코드가 삽입됩니다
필요한 정보
  • 이벤트 아이디 (eventID)
  • 섹션 제목
  • 댓글 작성 안내 문구 (선택)

작가 소개

화면 보기 ↗

이벤트에 작가를 소개하거나, 신간알림 버튼·저자파일 버튼을 추가할 때 사용합니다. 작가 프로필 없이 신간알림 버튼만 단독으로 추가하는 것도 가능합니다.

구성 요소
작가 프로필
역할 태그(글 / 그림 / 번역 등), 이름, 프로필 사진, 소개 글로 구성
신간알림 버튼
독자가 해당 저자의 신간을 알림 신청할 수 있는 버튼. 단독 추가도 가능
저자파일 버튼
알라딘 저자파일 페이지로 연결되는 버튼. 작가 ID 필요
필요한 정보
  • 역할 태그 (글 / 그림 / 번역 등)
  • 작가 이름
  • 작가 ID — 저자파일·신간알리미 연결에 사용
  • 프로필 사진 URL (선택)
  • 작가 소개 글
작가 ID란?
알라딘 저자파일 URL의 숫자 ID. 여러 작가 추가 가능.

신간알림 풀세트

화면 보기 ↗

신간 단행본 이벤트 페이지를 위자드 방식으로 한 번에 완성하는 모듈. 단계별로 안내에 따라 입력하면 이벤트 전체 코드가 자동 생성됩니다.

제작 순서
  1. 책 표지 이미지 URL을 입력합니다. 단일 표지 또는 여러 권 선택 가능 최대 4권
  2. 상단 디자인(히어로)을 설정합니다 — 모바일·PC 배경 이미지 URL, 배경색, 타이틀 입력
  3. 책 소개 텍스트를 입력합니다
  4. 아래 섹션은 체크박스로 포함 여부를 선택할 수 있습니다
    • 추천의 말 — 추천 문구·추천인·직함 입력
    • 만든 사람들 — 저자·역할 태그·소개 입력
    • 이벤트 안내 — 이벤트 안내 문구 입력
    • 댓글 — 댓글 이벤트 아이디(eventID) 입력
  5. 코드완료를 누르면 전체 이벤트 코드가 완성됩니다
필요한 정보
  • 책 표지 이미지 URL — 단일 또는 여러 권
  • 히어로 배경 이미지 — 모바일·PC 각각
  • 책 소개 텍스트
  • 추천의 말 — 추천인·직함 포함 (선택)
  • 작가 정보 — 이름·ID·프로필·소개 (선택)
  • 이벤트 안내 문구 (선택)
  • 댓글 이벤트 아이디 eventID (선택)

AI 배경 이미지 프롬프트 생성

화면 보기 ↗

AI 이미지 생성 툴을 사용할 때 프롬프트를 뭐라고 적어야 할지 막연하게 느껴질 때 유용합니다. 분위기·스타일·컬러를 선택하면 Midjourney·ChatGPT·Gemini에 바로 붙여넣을 수 있는 프롬프트가 자동으로 완성됩니다.

일반 모드 사용 방법
  1. 메인 컨셉·배경 스타일·분위기·컬러를 선택합니다
  2. 주요 오브젝트 위치를 선택합니다 — 좌측·우측·좌우 양 끝·중앙 등 원하는 배치 지정 가능 선택
  3. 고급 설정을 열면 더 상세한 연출을 직접 입력할 수 있습니다 선택
    • 추가 요청 — 원하는 분위기·연출을 자유롭게 기술
    • 금지 요소 — 기본값이 자동 입력되어 있으며, 추가로 제외하고 싶은 요소가 있으면 직접 작성
  4. 원하는 이미지 사이즈를 선택하고 프롬프트 복사하기를 클릭합니다
  5. ChatGPT·Gemini 등 이미지 생성 검색창에 프롬프트를 붙여넣으면 완성됩니다
참조 모드 사용 방법
  1. 상단의 참조 모드 토글을 켭니다 — 참고할 이미지의 분위기를 살려 배경을 생성할 때 사용
  2. 레이아웃을 선택합니다
    • 중앙 비우기
    • 중앙 채우기
  3. 핵심 요청을 입력합니다 선택 — 특정 이미지의 공간·패턴 등을 살려달라거나 포인트 요소만 활용해달라는 식의 구체적인 요청을 자유롭게 작성합니다
  4. 이미지 사이즈를 선택하고 프롬프트 복사하기를 클릭합니다
  5. 이미지 생성 툴에 복사한 프롬프트를 붙여넣고, 참고 이미지를 함께 첨부합니다
